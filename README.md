This app lets you extract structured data from PDF invoices using Google's Gemini API. Built with Streamlit for a user-friendly interface.

## Features
- Upload multiple PDFs
- Extract fields like invoice number, dates, amounts, etc.
- Download results as CSV
- Gemini AI-powered parsing

```bash
pip install -r requirements.txt
streamlit run app.py
